## Start Project

- Copy file .env.example to .env
- Add UNSPLASH_ACCESS_KEY to .env file
- Add Credit-nails DB_DATABASE / DB_USERNAME / DB_PASSWORD to .env file
- composer update
- npm i && npm run dev
- php artisan migrate:refresh --seed
- php artisan storage:link
